# SAKO Verification Checklist

The SAKO battery page uses the official SAKO catalogue matrix for the seven Solareco priority lithium battery models only. The local catalogue files remain in `reference-docs/`, which is ignored by Git and not served publicly.

| Check | Status | Notes |
| --- | --- | --- |
| Product category corrected | Complete | SAKO battery product category is `Lithium Batteries`, not `Solar Inverters`. |
| Public product-card name corrected | Complete | Product name is `SAKO Li-Sun Lithium Batteries`. |
| Secondary label added | Complete | Secondary label is `Energy Storage Batteries`. |
| Official source used | Complete | Source is `2026-03 SAKO Solar Catalogue 220V`, printed catalogue pages `25–30`. |
| Private local catalogue exposure | Complete | No private `reference-docs/` PDF is linked or copied into `public/`. |
| 320Ah model excluded | Complete | No SK-25.6V 320Ah or SK-51.2V 320Ah model is included in the public matrix. |
| 400Ah model excluded | Complete | No SK-25.6V 400Ah or SK-51.2V 400Ah model is included in the public matrix. |
| 640Ah model excluded | Complete | No SK-51.2V 640Ah model is included in the public matrix. |
| 600Ah model included | Complete | SK-51.2V 600Ah is included in the public matrix. |
| Technical values guessed | Complete | No estimated or guessed technical values are used in the public matrix. |
